## Versions:

Angular CLI: 14.1.2
Node: 16.10.0
Package Manager: npm 7.24.0

## Reference:

https://www.assemblyscript.org/getting-started.html#setting-up-a-new-project

## Steps to run:

- npm init
- npm install --save-dev assemblyscript
- npx asinit .
- npm install
- npm run asbuild
- npm test (optional)
- npm start
